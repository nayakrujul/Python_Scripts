Tables

Run the 'main.py' file to open 'index.html'.

From there, add and delete rows using those buttons,
and also edit the cells in the table by just clicking
and entering your desired text. Change the title text
by clicking on 'change text', entering desired text,
then again clicking 'change text' to save changes.
Change the colour of the title by typing it in and
clicking 'change'. Change the table border by typing
it in and clicking 'change'. Change the page background
by typing it in and clicking 'change'.

No PIP installations necessary.
It will open the file in your default browser.

Written in Python 3.9 / HTML5
